# Luke Barrier Dotfiles

## Prerequisites

### MacOS

```bash
brew install stow
```

### Linux

#### Debian

```bash
sudo apt-get install stow
```

#### Arch

```bash
sudo pacman -S stow
```

## Install

```bash
git clone https://github.com/lukebarrier/dotfiles ~/dotfiles --depth=1 && ~/dotfiles/stow.sh && zsh -c "clear"
```
