# Luke Barrier Dotfiles

## Prerequisites
### MacOS
```bash
brew install stow
```

### Linux

#### Debian
```bash
sudo apt-get install stow
```

#### Arch
```bash
sudo pacman -S stow
```

## Install
```bash
cd ~ && gh repo clone dotfiles && cd dotfiles && chmod +x stow.sh && ./stow.sh && cd ~ && zsh
```
